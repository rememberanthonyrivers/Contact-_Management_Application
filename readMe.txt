***/ Contact Management Application /***

This is a simple contact management application that allows users to create, read, update, 
and delete (CRUD) contact information. The backend is built with Flask, a lightweight WSGI 
web application framework in Python, and the frontend is built with React, a JavaScript 
library for building user interfaces.

*** Features ***

* List all contacts
* Add a new contact
* Update an existing contact
* Delete a contact
* View contact details

*** Prerequisites ***

Before you begin, ensure you have met the following requirements:

Python 3.8 or above
Node.js and npm
Git (for cloning the repository)

*** Usage ***
After starting both the backend and frontend servers, you can use the application to manage
contacts. The interface is straightforward, providing forms and lists for interacting with contact data.
